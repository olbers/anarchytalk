About

AnarchyTalk is a Anarchy Online™ chat client for Android devices. It is written by Novagen (RK1), in cooperation with AO-Universe, and based on the Java AO Chat API by Losobie. The icon of the application is created by Ukblizzard (RK1).

Item look-ups are done on the Xyphos website.
